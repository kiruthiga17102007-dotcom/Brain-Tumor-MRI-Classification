**BRAIN TUMOR MRI CLASSIFICATION**

This project uses Deep Learning (CNN) to classify MRI brain scan images into four categories:

- Glioma
- Meningioma
- Pituitary
- No Tumor

**DATASET**
- Training Images: 5600
- Testing Images: 1600

**TECHNOLOGIES USED**
- Python
- TensorFlow
- Keras
- Google Colab

**RESULT**
- Training Accuracy: 98.75%
- Testing Accuracy: 87.56%
